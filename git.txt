- node_modules : 프로젝트에서 사용하는 외부 라이브러리들이 저장되는 공간

- public : 정적 자원 (파일)들이 저장되는 공간

- src : 실제 소스코드가 저장되는 공간

- .gitignore : 해당 파일이 있는 위치에서 git 작업을 할 때 git 작업에서 무시되는 파일을 관리하는 파일

- package.json : npm 관련 명령어, 해당 프로젝트에 대한 설명, 해당 프로젝트에서 사용하는 패키지 (모듈, 라이브러리 등)가 정의가 되어 있음

- package-lock.json : package.json이 적용된 것들을 명시한 파일

- README.md : github에 README를 적용하기 위한 파일

- tsconfig.json : typescript를 적용하기위한 설정
